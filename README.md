# News-Scraper
All the News That's Fit to Scrape (from theverge.com/tech)

* Web scraper built using:
    * Javascript
    * express
    * express-handlebars
    * mongoose
    * cheerio
    * axios
    * MongoDB
    * mLab remote MongoDB for deployment with Heroku

Link to deployed page: [Heroku](https://sleepy-badlands-27283.herokuapp.com/)
